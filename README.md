# Hospital Analytics – Power BI Dashboard

This project presents an **interactive Power BI dashboard** developed to analyze hospital performance, patient statistics, and healthcare metrics. The dashboard enables users to explore hospital data through key KPIs, visuals, and dynamic filters.

## Dashboard Overview

The report contains a single analytics page that highlights important healthcare insights.

### Key KPIs

* Total Revenue
* Total Patients
* Bed Occupancy %
* Total Emergency Cases
* Average Ratings
* Average Length of Stay

### Interactive Filters

Users can filter the dashboard using:

* Insurance Provider
* Doctor
* Year
* Month

### Visualizations

* **Donut Chart:** Percentage of patients affected by different diseases
* **Column Chart:** Total patients per year
* **Doctor Analysis (Bar Chart):** Total patients handled by each doctor

### Data Tables

* **Patient Details Table:** Name, Age, Medical Condition, Doctor, Date of Admission, Discharge Date, Length of Stay, Test Results, Bill Amount, and Ratings
* **Gender Summary Table:** Total Patients, Emergency Cases, Bed Occupancy, and Total Revenue by Gender

## Purpose

The dashboard helps analyze hospital operations, monitor patient statistics, and evaluate healthcare performance through interactive data exploration.

## Tools Used

* Power BI
* DAX Measures
* Interactive Filters & Visualizations
